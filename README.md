# Spam Email Detector

A beginner machine learning project that detects whether an email is spam or not spam.

## Features
- Uses TF-IDF for text processing
- Uses Naive Bayes classifier
- Shows model accuracy
- Allows users to test emails interactively

## Technologies Used
Python
Scikit-learn
Machine Learning

## How to Run

pip install -r requirements.txt
python spam_detector.py

Type an email message and the model will predict whether it is spam or not spam.
